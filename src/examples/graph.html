📝 Instructions

1. Click Subscribe to start listening for notifications for the given user ID.
2. Open another tab with GraphQL Playground (if enabled) or use Postman.
3. Run a mutation that publishes an event to the NOTIFICATIONS channel.
4. If you don't have a mutation yet, you can add one like this to your resolver: 

@Mutation(() => Boolean)
async sendNotification(
    @InjectPubSub() pubsub: any,
    @Arg('userId') userId: string,
    @Arg('message') message: string
) {
    await pubsub.publish('NOTIFICATIONS', {
        id: Date.now().toString(),
        userId,
        name: 'System',
        email: 'no-reply',
        message
    });
    return true;
}

Then call it with:

mutation {
  sendNotification(userId: "1", message: "Hello via subscription!")
}
